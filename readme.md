# 项目启动

```shell
uvicorn main:app --reload
```

# 项目管理

使用conda管理Python版本
使用Poetry管理Python包

```shell
pip install poetry

# 检查版本
poetry --version

```

| 名称      | 功能                                    |
|---------|---------------------------------------|
| new     | 创建一个项目脚手架，包含基本结构、pyproject.toml 文件    |
| init    | 基于已有的项目代码创建 pyproject.toml 文件，支持交互式填写 |
| install | 安装依赖库                                 |
| update  | 更新依赖库                                 |
| add     | 添加依赖库                                 |
| remove  | 移除依赖库                                 |
| show    | 查看具体依赖库信息，支持显示树形依赖链                   |
| build   | 构建 tar.gz 或 wheel 包                   |
| publish | 发布到 PyPI                              |
| run     | 运行脚本和代码                               |
| shell   | 激活虚拟环境                                |
